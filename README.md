Este módulo é dedicado ao aprofundamento do conceito central da biblioteca: os Componentes.

Originado do curso "Como funcionam os componentes", este repositório explora em detalhes como estruturar, reutilizar e comunicar componentes de forma eficaz. Aborda a diferença e a aplicação prática entre Componentes de Função (Funcionais) e Componentes de Classe, a importância da reutilização para manter o código limpo (Princípio DRY - Don't Repeat Yourself) e como o fluxo de dados unidirecional (props) garante a previsibilidade das aplicações React.

Tópicos Chave Explorados:

Composição e hierarquia de componentes.

Propriedades (props) e a comunicação parent-child.

Reuso de código e criação de componentes independentes.

Compreender profundamente os componentes é a chave para dominar o desenvolvimento com React.
